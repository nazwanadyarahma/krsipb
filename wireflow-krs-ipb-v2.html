+
GOOD Form sederhana, hanya 2 field → cognitive load minimal.
!
ISSUE Tidak ada indikator "Lupa Password" yang jelas — pengguna panik ketika lupa.
i
Alt Path: Login gagal → tampilkan pesan error spesifik (bukan generic "salah username/password").
SCR 2Home KRS (Updated)
★
NEW Status Akademik strip — konfirmasi visual bahwa SPP lunas + perwalian selesai sebelum isi KRS.
★
NEW Batas SKS bar dengan progress bar — pengguna tahu kuota sebelum memilih MK, mengurangi error.
★
NEW Tombol "Cek Biodata" di panel Data Mahasiswa — akses cepat verifikasi data diri tanpa navigasi panjang.
!
ISSUE 15 poin pengumuman masih tanpa prioritas → rekomenasi: badge KRITIS / PENTING / INFO.
